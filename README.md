# 🔍 GitHub Profil Analyzer

GitHub kullanıcı adını girerek profilini analiz eden AI destekli uygulama.

## 🚀 Özellikler
- Programlama dili dağılımı grafiği
- Geliştirici tipi tahmini (AI, Backend, Frontend)
- Profil skoru hesaplama
- Kişiselleştirilmiş öneriler

## 🛠️ Teknolojiler
- Python
- Gradio
- GitHub API
- Plotly

## ▶️ Çalıştırma
```bash
pip install gradio requests plotly
python app.py
